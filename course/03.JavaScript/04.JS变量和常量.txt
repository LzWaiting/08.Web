JS 变量与常量
	1. 变量
		1. 创建变量
			1. 先声明，后赋值，使用关键字 var 进行声明
				声明：var 变量名;
				赋值：变量名 = 值;
			2. 声明的同时赋值
				var 变量名 = 值;
		2. 使用注意
			1. 声明变量时，关键字var可以省略，不建议省略，容易出问题
			2. 变量如果只声明，未赋值，
				var a;
				变量默认值为 undefined
		3. 变量名的命名规范
			1. 不能与JS中的关键字和保留字冲突（var default break class function for while do...while name）
			2. 自定义的变量名可以由数字，字母，下划线，$组成，不能使用数字开头
			3. 变量名称严格区分大小写
			4. 命名尽量有意义一些，做到见名知意
			5. 多个单词组成变量名，采用小驼峰标识
		4. 变量的使用方式
			1. 赋值
				赋值符号：=
				作用：将赋值符号右边的值赋值给左边的变量
				var a = 10;
			2. 访问
				1. 直接输出
					console.log(a);
					document.write(a);
				2. 变量出现在赋值符号右边，表示访问变量的值
					var age = 10;
					age = age + 20;
				注意：赋值符号的左边只能是变量
	2. 常量
		1. 已经声明就不允许再被修改的数据就是常量
		2. 语法：
			const 常量名 = 值;
			注意：
				1. 常量在声明的同时必须赋值
				2. 为了与变量名区分，常量名一般采用全大写的方式
	3. 操作小数位
		保留小数点后 n 位
		toFixed(n);
		使用：
			var num = 3.1415926;
			var res = num.toFixed(2);
