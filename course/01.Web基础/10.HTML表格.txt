表格
	
	1. 语法：
		1. 标签介绍
			表格标签：	<table></table>		--> table
			行标签：		<tr></tr>			--> table row
			单元格标签：	<td></td>			--> table data	
		2. 创建顺序
			1. 创建表格标签
			2. 在表格标签中嵌套行标签，每一个tr就代表一行
			3. 在行标签中创建单元格标签，用来存放数据
			et:
				见 09-table.html	
	
	2. 标签属性
		1. table 属性
			1. border:设置边框，取值以px为单位的数值（px可以省略）
			2. width:设置宽度
			3. height:设置高度
			4. align:设置表格在其父元素中的水平对齐方式
			5. cellpadding:设置单元格的内边距（内容与边框之间的距离），取值为px单位数值
			6. cellspacing:设置单元格的外边距（单元格与单元格之间的距离，单元格与表格边框之间的距离）
			7. bgcolor:设置表格的背景颜色，取值可以是英文的颜色名称
		2. tr 属性
			1. bgcolor:设置当前行的背景颜色
			2. align:设置当前行中内容的水平对齐方式
			3. valign:设置当前行中内容的垂直对齐方式，取值top/middle/bottom,默认垂直居中
		3. td 属性
			1. width:设置单元格宽度
			2. height:设置单元格高度
			3. align:单元格内容的水平对齐方式
			4. valign:单元格内容的垂直对齐方式
			5. bgcolor:单元格的背景颜色
		4. th 属性
			作用：一般用于首行单元格，自带加粗居中效果
			用法：与td 标签一致

	3. 单元格合并（重点）
		是单元格独有的属性 
		colspan		--> colume span 	跨列合并
		rowspan		--> row span 		跨行合并
		1. 单元格跨列合并
			从当前单元格的位置开始，横向向右合并几个单元格
			colspan='n'	跨n列合并（包含自身）
		2. 单元格跨行合并
			从当前单元格的位置开始，纵向向下合并几个单元格
			rowspan='n'	跨n行合并（包含自身）
		注意：一旦发生单元格合并
			跨列合并，要删除当前行中多余的单元格
			跨行合并，要删除其后行中多余的单元格
			始终保持表格结构的完整性
		示例：见 10-colspan.html

	4. 行分组
		表格结构
		1. 行分组
			允许将表格中的一行或者若干行划分为一组，便于管理
		2. 语法
			1. 表头行分组
				<thead>
					<tr>
						<td></td>
					</tr>
					<tr></tr>
				</thead>
			2. 表尾行分组
				<tfoot>
					<tr>
						<td></td>
					</tr>
				</tfoot>
			3. 表主体信息
				<tbody>
					<tr>
						<td></td>
					</tr>
				</tbody>
			注意：<thead>/<tfoot>/<tbody>可以省略，默认情况下，所有的行都会被自动添加到<tbody>中
		示例：
			见 11-tbody.html
		注意：
			如果需要手动添加行分组，建议按照<thead>--<tfoot>--<tbody>的顺序书写，便于网页显示效率
